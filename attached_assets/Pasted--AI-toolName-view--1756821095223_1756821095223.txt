## 项目配色方案详细说明

根据对各个页面代码的分析，以下是每个页面的具体配色方案：

### 1. 登录页面 (LoginPage.vue)
**背景色调：**
- 页面背景：`bg-gray-50` (浅灰色背景)
- 表单卡片：`bg-white` (白色卡片) + `shadow-medium` (中等阴影)

**文字颜色：**
- 主标题：`text-gray-900` (深灰色)
- 副标题：`text-gray-600` (中灰色)
- 标签文字：`text-gray-900` (深灰色)
- 占位符：`placeholder-gray-500` (中等灰色)

**交互元素：**
- 输入框边框：`border-gray-300` → `focus:border-primary-500` (灰色到蓝色)
- 登录按钮：`bg-gradient-to-r from-primary-500 to-primary-600` (蓝色渐变)
- 错误提示：`bg-error-50` + `border-error-200` + `text-error-600` (红色系)
- 管理员提示：`bg-primary-50` + `border-primary-200` (浅蓝色背景)

### 2. 欢迎页面 (WelcomePage.vue)
**背景色调：**
- 页面背景：`bg-gradient-to-br from-gray-50 via-blue-50 to-gray-50` (灰色到浅蓝色渐变)
- 用户卡片：`bg-white/90 backdrop-blur-sm` (半透明白色磨砂玻璃)

**主要元素：**
- Logo标题：`bg-gradient-to-r from-primary-500 to-primary-600` (蓝色渐变文字)
- 功能图标：`text-primary-500` (蓝色图标)
- 开始学习按钮：`bg-gradient-to-r from-primary-500 to-primary-600` (蓝色渐变背景)

### 3. 学习页面 (StudyPage.vue)
**背景色调：**
- 页面背景：`bg-gradient-to-br from-gray-50 to-blue-50` (灰色到浅蓝色渐变)
- 左侧导航：`bg-white/90 backdrop-blur-sm` (半透明白色磨砂玻璃)
- 内容卡片：`bg-white` (纯白色背景)

**导航元素：**
- 选中段落：`bg-primary-50` + `border-primary-200` (浅蓝色背景)
- 进度条：`bg-primary-500` (蓝色进度条)
- 操作按钮：`bg-gradient-to-r from-primary-500 to-primary-600` (蓝色渐变)

### 4. 训练页面 (TrainingPage.vue)
**背景色调：**
- 页面背景：`bg-gradient-to-br from-gray-50 to-blue-50` (灰色到浅蓝色渐变)
- 顶部导航：`bg-white/90 backdrop-blur-sm` (半透明白色磨砂玻璃)
- 内容卡片：`bg-white` (纯白色背景)

**状态颜色：**
- 正确答案：`text-success-500` (绿色)
- 错误答案：`text-error-500` (红色)
- 完成状态：`bg-success-50` + `border-success-200` + `text-success-600` (绿色系)
- 主要按钮：`bg-gradient-to-r from-primary-500 to-primary-600` (蓝色渐变)

### 5. AI聊天页面 (AiChatPage.vue)
**背景色调：**
- 页面背景：`bg-gray-50` (浅灰色背景)
- 顶部导航：`bg-white` (白色背景)
- 消息卡片：`bg-white` (白色背景)

**聊天元素：**
- AI头像：`bg-primary-50` + `border-primary-200` + `text-primary-500` (浅蓝色背景)
- 用户头像：`bg-primary-100` + `border-primary-300` + `text-primary-600` (蓝色背景)
- 用户消息：`bg-primary-50` + `border-primary-200` (浅蓝色背景)
- API状态指示：`bg-success-500` / `bg-error-500` / `bg-warning-500` (绿/红/橙色)

### 6. 复述页面 (ParaphrasePage.vue)
**背景色调：**
- 页面背景：`bg-gradient-to-br from-gray-50 to-blue-50` (灰色到浅蓝色渐变)
- 顶部导航：`bg-white/90 backdrop-blur-sm` (半透明白色磨砂玻璃)
- 内容卡片：`bg-white` (纯白色背景)

**录音元素：**
- 录音按钮（未录音）：`bg-primary-50` + `border-primary-500` + `text-primary-500` (蓝色系)
- 录音按钮（录音中）：`bg-red-100` + `border-red-500` + `text-red-500` (红色系)
- 评分进度条：`bg-gradient-to-r from-red-500 via-yellow-500 to-green-500` (红黄绿渐变)

**评估结果：**
- 各维度评分：蓝色(`bg-blue-50`)、绿色(`bg-green-50`)、紫色(`bg-purple-50`)、橙色(`bg-orange-50`)
- 优点提示：`bg-success-50` + `border-success-500` (绿色系)
- 改进建议：`bg-warning-50` + `border-warning-500` (橙色系)

### 整体配色特点
1. **主色调**：以蓝色系(`primary-*`)为主题色，贯穿所有页面
2. **背景层次**：浅灰色到浅蓝色渐变背景 + 白色/半透明白色卡片
3. **功能色彩**：绿色(成功)、红色(错误)、橙色(警告)用于状态提示
4. **视觉效果**：磨砂玻璃效果(`backdrop-blur-sm`)、渐变背景、阴影层次
5. **文字层次**：深灰色(`gray-900`)主文字、中灰色(`gray-600`)辅助文字、浅灰色(`gray-500`)占位符
